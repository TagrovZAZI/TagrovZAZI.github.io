Всем привет!

Меня зовут Александр и это мой первый сайт-порфтолио.
Сайт выполнен с использованием html, css и js.
Html позиционирование и css стилизация выполнены в ручную, однако js файлы были внедрены из разных источников. Так, к данному сайту подключены следующие js файлы: Parallax эффект из видео урока "WebDesign Master" на "YouTube", плавный скролл от "gsap" и IntersectionObserverAPI (активация анимации при появлении в видимую область).

Биография

Мой путь в веб-разработке начался в 2022 году, когда я решил открыть свой первый бизнес в городе Якутске. Я решил попробовать создавать 3D-туры для квартир. После изучения процесса создания 3D-туров я решил создать одностраничный сайт для своей будущей деятельности. Я выбрал написание сайта вручную, чтобы получить навык и лучше разобраться в системе разработки сайтов. Я освоил основы HTML, позиционирования элементов и стилизации с помощью CSS. Затем я скачал простой макет сайта с GitHub и настроил его под свои нужды. Так я создал свой первый сайт под названием "tagrov360".
К сожалению, я не получил достаточного отклика от владельцев недвижимости на свои 3D-туры и решил прекратить развитие этого бизнеса. Этот опыт побудил меня заинтересоваться веб-разработкой. В результате этого интереса я создал данный портфолио-сайт. В настоящее время я начинаю изучать основы JavaScript и планирую освоить такие библиотеки, как React и Three.js в будущем.
